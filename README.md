# ZestGo-Food-Delivery-App
UI/UX Design of a modern food delivery application created in Figma.


## Overview
ZestGo is a modern food delivery application designed to provide a seamless food ordering experience.


## My Role
UI/UX Designer


## Tools Used
- Figma
- Canva


## Features
- Restaurant Discovery
- Search & Filter
- Cart Management
- Secure Checkout
- Order Tracking
